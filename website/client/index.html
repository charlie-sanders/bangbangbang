bang

BANG is an inline ticketing system that allows you to !tag your comments and assign them a ^priority, then displays them with a text report or a web report for easily keeping track of TODO's and the like.

To Install:

curl http://www.quick-ticket.com | sh

Example Usage:

var SocketHandler = function () {

// Fix this to read a config file on init !charlie ^5
this.host = "localhost";

};

The !tag is to categorize the comment , and the ^N is to mark the priority [0-9]
